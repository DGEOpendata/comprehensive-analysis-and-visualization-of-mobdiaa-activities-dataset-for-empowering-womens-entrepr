# Comprehensive Analysis and Visualization of Mobdiaa Activities Dataset

## Overview
The "Mobdiaa Activities Dataset" is a valuable resource that provides insights into the diverse entrepreneurial activities supported under the Mobdiaa initiative in Abu Dhabi. This dataset is a key asset for researchers, policymakers, and other stakeholders looking to understand and promote women's entrepreneurship and small businesses.

This project demonstrates how to analyze and visualize the dataset to extract actionable insights. By leveraging the dataset's bilingual content and focusing on a niche theme, we create a user-friendly analytical platform that empowers stakeholders to make informed decisions.

## Features
- **Data Exploration:** Filter and sort through the dataset using various parameters such as activity type, language, and geographic location.
- **Dynamic Visualizations:** Generate interactive graphs and charts to understand trends and patterns.
- **Data Export:** Extract specific subsets of the dataset for further analysis.
- **Community Engagement:** Share insights and findings through integrated collaboration tools.

## Prerequisites
- Python 3.8+
- Pandas library
- Matplotlib library
- Seaborn library
- An Excel reader (such as Microsoft Excel or Google Sheets) to view the dataset.

## Installation
1. Clone this repository:
    
    git clone https://github.com/your-repository/mobdiaa-dataset-analysis.git
    cd mobdiaa-dataset-analysis
    
2. Install the required Python packages:
    
    pip install pandas matplotlib seaborn
    

## Usage
1. Place the dataset file (`DL02-Mobdiaa-Activities-ADRA-OD-006-AMO_0.xlsx`) in the `data` folder.
2. Run the script `analyze_mobdiaa.py` to perform data analysis and generate visualizations:
    
    python analyze_mobdiaa.py
    
3. View the generated bar chart to understand the distribution of activity categories.
4. Check the `output` folder for any filtered data files, such as `tailoring_activities.xlsx`.
